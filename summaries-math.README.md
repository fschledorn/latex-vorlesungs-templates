Math summaries (summarystyle-math.cls)
------------------------------------

Purpose

This class is tuned for mathematics lectures where you want strict definitions, lemmata, theorems and proofs.

Usage

```tex
\documentclass{path/to/summarystyle-math}
```

Notes

- German-style theorem names are used (Definition, Satz, Lemma, Korollar, Beweis)
- Important statements are framed for emphasis
- Includes the standardized macro set used across the repo

See the top-level `README.md` for attribution and license info.
