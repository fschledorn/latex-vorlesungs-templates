Book / script summaries (script-style.cls)
-----------------------------------------

This class is intended for longer summaries or a small 'script' document.

Usage

```tex
\documentclass{path/to/script-style}
```

Notes

- Report/book style layout
- Good for longer write-ups or multi-chapter notes
- Uses the same standardized macros as the other classes for consistency
